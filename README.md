# 🚀 Factify — Cut Through the Noise

Factify is an AI-powered web application designed to verify the authenticity of information in real-time.  
In an era flooded with misinformation, Factify helps users quickly determine whether a claim is **Verified**, **Fake**, or **Misleading**.

---

## 🧠 What It Does
Factify analyzes user-provided text (news, claims, or statements) and:
- Identifies the core factual claim  
- Evaluates it against real-world knowledge  

---

## 🏷️ Classification System
- ✅ **Verified** — factually correct  
- ❌ **Fake** — incorrect or false  
- ⚠️ **Misleading** — partially true or lacking context  

---

## 📊 Output Provided
- Confidence score (0–100)  
- Simple explanation  
- Key reasoning signals  

---

## ⚡ Why Factify?
Misinformation spreads fast — faster than people can verify it.  

Factify is built to:
- 🛑 Stop fake news before it spreads  
- 📊 Promote informed decision-making  
- ⚡ Deliver instant, easy-to-understand verification  

---

## 🛠️ Tech Stack
- **Frontend:** React + Vite  
- **Backend:** Node.js + Express  
- **AI Integration:** LLM-based analysis (Groq / OpenAI compatible)  
- **Authentication:** JWT-based  

---

## 🎨 Features
- 🔍 Real-time claim analysis  
- 📊 Confidence scoring system  
- 🧾 Detailed reasoning signals  
- 🌍 Multi-language support  
- 🎨 Clean modern UI (Glassmorphism design)  
- 🔐 User authentication & history tracking
